# **FakeInterior**

***Shader de interiores falsos con paralaje en URP (Amplify). Eficiente y personalizable.***

</header>

## Preview

![Preview](https://github.com/user-attachments/assets/33d05c9a-dcc1-4a13-85dc-44c9fede212d)

## Requirements

* Unity: Unity 6
* Render Pipeline: Universal Render Pipeline
* Required packages/tools:

  * Amplify Shader Editor

## Usage

Apply the FakeInterior shader/material to building window geometry and adjust the exposed parameters to control the fake room depth, tiling, parallax intensity, and visual variation.
